# Iris Flower Classification

## Project Description

This project is developed as part of the CodeAlpha Data Science Internship.

The model classifies Iris flowers into three species:

Setosa
Versicolor
Virginica

using Machine Learning.

## Technologies Used

Python
Pandas
Matplotlib
Scikit-learn

## Machine Learning Algorithm

Random Forest Classifier

## Features

Load Iris Dataset
Data Analysis (EDA)
Missing Value Checking
Data Visualization
Histogram
Scatter Plot
Train/Test Split
Model Training
Prediction
Accuracy Evaluation
Confusion Matrix
Classification Report

## Project Structure

app.py
requirements.txt
README.md

## How to Run

1. Install the required libraries:

pip install -r requirements.txt

2. Run the project:

python app.py

## Output

The model predicts the species of an Iris flower based on:

Sepal Length
Sepal Width
Petal Length
Petal Width

The project also displays:

Histogram
Scatter Plot
Accuracy Score
Confusion Matrix
Classification Report
